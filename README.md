# projetobi-Analise-Qualide-De-Vida-Celular
Este projeto consiste em uma análise exploratória de dados (EDA) utilizando um dataset público do Kaggle sobre tempo de uso de celular, sono, estresse e produtividade.  O objetivo foi investigar possíveis relações entre o tempo de tela e indicadores de qualidade de vida, utilizando o Power BI para modelagem e visualização dos dados.
